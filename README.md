# Proyecto SEMINARIO ML - Trabajo Práctico 5

## Estructura del proyecto

- `tp_en_cliente/`: contiene los archivos para ejecutar directamente en el navegador.
- `tp_servidor_js/`: contiene la aplicación del servidor con Node.js.

## Uso de `tp_en_cliente`

Dentro de la carpeta `tp_en_cliente` solo se debe ejecutar el archivo `index.html`.

1. Abrir la carpeta `tp_en_cliente`.
2. Abrir `index.html` en un navegador web.

> No es necesario instalar dependencias ni ejecutar comandos adicionales para esta parte.

## Uso de `tp_servidor_js`

Para ejecutar el servidor en `tp_servidor_js`, sigue estos pasos:

1. Clona el repositorio en tu máquina.
2. Muévase a la carpeta del repositorio.
3. Entre en el directorio `tp_servidor_js`.
4. Instale las dependencias con el comando:

```bash
npm i
```

5. Ejecute el servidor con:

```bash
npm run dev
```

6. En la salida de la consola aparecerá una dirección URL. Abra esa dirección en su navegador para acceder a la aplicación.

> Asegúrese de estar en el directorio `tp_servidor_js` antes de ejecutar `npm i` y `npm run dev`.
